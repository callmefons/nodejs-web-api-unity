### Setup

##### Main Repository
1. `git clone https://github.com/cfcfon/NodeJS-WebAPI-Unity.git`
2. `cd NodeJs-WebAPI-Unity`
3. `copy a directory WebAPI Client to Unity3D Engine`
4. `npm install`
5. `node index.js`
